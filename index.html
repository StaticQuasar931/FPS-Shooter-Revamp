Controls

  • WASD move, Mouse aim
  • Left Click shoot
  • V melee when enemies get close
  • Shift run (uses stamina)
  • Space jump
  • R reload
  • F shop toggle, Esc pause or close shop

Notes

  • Click Start. Your mouse will lock into the game.
  • If you tab out, the game pauses. Click Resume to continue.
  • Run with a local server for testing.
If textures do not load locally, use python -m http.server and open http://localhost:8000.
